Integrated Assignment Environment (IAE)

User manual — CE316 project. Offline desktop tool for batch-evaluating student programming submissions.

1. Getting started

Install IAE from the Windows installer, then launch it from the desktop shortcut. Java 21 must be installed on the target machine.

2. Language configurations

Open Configuration → Manage Configurations to create profiles for C, Java, Python, etc. Each profile defines:

  • Compiler or interpreter path
  • Compile arguments and expected source filename
  • Run command and timeout

Use Import / Export to share configurations between machines.

3. Projects

File → New Project links a configuration, submission ZIP folder, expected output file, and optional run arguments. Save the project to persist settings and results.

4. Running evaluation

Place student ZIP files in one folder (each archive should extract to a folder named with the student ID). Click Run Evaluation. IAE will unzip, compile, run, and compare outputs for every ZIP. Errors for one student do not stop processing for others.

5. Results

The results table shows compile status, pass/fail, and timing per student. Select a row to view error logs in the panel below.
